# Project Node: *Mastery* Overview

This is the main project node for *Mastery* by Robert Greene. This book outlines a path to achieving mastery through discovering your life's purpose, undergoing an apprenticeship, entering a creative-active phase, and ultimately reaching mastery.

## Key Themes
- [[Life's Task]]
- [[Apprenticeship]]
- [[Creative-Active Phase]]
- [[Mastery]]

Each theme represents a distinct stage or component in the journey toward mastering a skill or field.
