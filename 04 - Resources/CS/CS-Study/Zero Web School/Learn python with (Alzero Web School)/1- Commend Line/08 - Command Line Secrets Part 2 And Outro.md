## Definition of this lesson: (Make shortcuts of commends, Explorers the folders & files, See the username, see the connection of web or device)

Windows:
- `explorer `: (Windows File Explorer command), Used to open any file or folders, If you want to open your path folder write  `explorer . `, And also it used to open the site web.
- `ipconfig `: (Internet Protocol Configuration), Used to show you network configuration like:
```Displays and updates your device's network configuration, such as:

IP ​​address
Gateway address
DNS address
Status of connected networks

// And it syntax is `ipconfig ` , For linux is `ifconfig `
```

- `whoami `: To see the username, Linux & windows.
- `ping `: (Packet Internet Groper), Used to see if the device or website is online or not, Without `https:// || http:// ` Just the domain name like `ping roadmap.sh `, Linux & windows.

Linux:
- `alias `: (Command Shortcut), Used to make an easy shortcuts alias and it syntax is: `alias shortcus=commend`. 
- `whoami `: To see the username, Linux & windows.
- `ping `: (Packet Internet Groper), Used to see if the device or website is online or not, Without `https:// || http:// ` Just the domain name like `ping roadmap.sh `, Linux & windows.
