# **`[...]` و `{...}` و `(...)`**

# #️⃣ **1) الأقواس المربعة: `[ ]`**

## ✦ وظيفتها: **Character Class**

تعني: “اختر واحد فقط من بين هذه المجموعة من الحروف”.

مثال:

```
[abc]
```

يطابق:

- a
    
- b
    
- c  
    فقط.
    

---

## ✦ أمثلة قوية:

### 1) أي رقم:

```
[0-9]
```

### 2) أي حرف كبير:

```
[A-Z]
```

### 3) أي حرف صغير:

```
[a-z]
```

### 4) حرف **ليس** رقمًا:

```
[^0-9]
```

(لأن ^ داخل الأقواس = نفي)

---

## ✔ عندما ترى `[ ]` فاعرف أنه يتم اختيار **حرف واحد فقط** من مجموعة.

---

# #️⃣ **2) الأقواس المعقوفة: `{ }`**

## ✦ وظيفتها: **Quantifier — تكرار عدد معين من المرات**

تعمل مع الشيء الذي قبلها.

### أمثلة:

### 1) حرف يتكرر **مرتين**:

```
A{2}
```

يطابق:

- AA
    

### 2) رقم يتكرر **3 مرات**:

```
[0-9]{3}
```

يطابق:

- 123
    
- 938
    
- 004
    

### 3) رقم يتكرر **من 2 إلى 4 مرات**:

```
[0-9]{2,4}
```

يطابق:

- 12
    
- 345
    
- 9087
    

### 4) رقم يتكرر **على الأقل 3 مرات**:

```
\d{3,}
```

---

## ✔ عندما ترى `{ }` فاعرف أنه يعني **كم مرة يظهر الحرف أو المجموعة**.

---

# #️⃣ **3) الأقواس العادية: `( )`**

## ✦ وظيفتها الأساسية: **Grouping (تجميع)**

تجمع عدة حروف معًا ليتم التعامل معها كـ **وحدة واحدة**.

### مثال:

```
(ab)+
```

هذا يعني:

- ab
    
- abab
    
- ababab  
    لأن `+` أصبح يكرر **المجموعة** وليس حرفًا واحدًا.
    

---

## ✦ وظيفتها الثانية: **Capturing Group**

تلتقط النص الذي طابقته حتى تستطيع استعماله.

مثال:

```python
re.search(r"(\d+)-(\d+)", "25-99")
```

المجموعة 1 = 25  
المجموعة 2 = 99

---

## ✦ أمثلة قوية:

### 1) تكرار كلمة كاملة:

```
(hello){2}
```

يطابق:

- hellohello
    

### 2) كلمة بها جزء اختياري:

```
colou?r
```

لكن لو كنا نبغى الجزء الاختياري يكون مجموعة:

```
colo(u)?r
```

### 3) استخراج جزء محدد:

```
(\d{4})-(\d{2})-(\d{2})
```

لتاريخ:

- 2024-11-03  
    (سنة – شهر – يوم)
    

---

## ✔ الـ `( )` تجعلك تتحكم في **مجموعة كاملة** وليس في حرف واحد.

---

# 🧠 خلاصة سريعة جدًا

|الرمز|اسمه|دوره|
|---|---|---|
|`[ ]`|Character Class|اختيار حرف واحد من داخل المجموعة|
|`{ }`|Quantifier|عدد مرات التكرار|
|`( )`|Grouping / Capturing|تجميع حروف للتعامل معها كوحدة واحدة|

---

# 📌 مثال واحد يجمع الثلاثة لتثبيت الفهم:

```
([A-Z]{2})-(\d{3,5})
```

### تفسيره:

- `[A-Z]{2}`  
    حرفين capital (لأن {} تكرر ما داخل [])
    
- `()`  
    يجمع الحرفين معًا كـ مجموعة 1
    
- `-`  
    حرف عادي
    
- `\d{3,5}`  
    رقم يتكرر من 3 إلى 5 مرات  
    (لأن {} تكرر الـ `\d`)
    
- `()`  
    مجموعة 2 للرقم
