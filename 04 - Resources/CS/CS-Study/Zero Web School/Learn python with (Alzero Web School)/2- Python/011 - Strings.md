# <font color="#646a73">011 - Strings</font>
---
## <font color="#ffc000">Explanation and method of use:</font>
#### Input:
```python
# -------------
# -- Strings --
# -------------

myStringOne = 'This is Single Quote'
myStringTwo = "This is Double Quotes"

print(myStringOne)
print(myStringTwo)

myStringThree = 'This is Single Quote "Test"'
myStringFour = "This is Double Quotes 'Test'"

print(myStringThree)
print(myStringFour)

myStringFive = '''First
Second 'Test' "Test"
Third'''

myStringSix = """First
Second "Test" \\\ 'Test'
Third"""

print(myStringFive)
print(myStringSix)
```
Output:
```
This is Single Quote
This is Double Quotes
This is Single Quote "Test"
This is Double Quotes 'Test'
First
Second 'Test' "Test"
Third
First
Second "Test" \\ 'Test'
Third
```
