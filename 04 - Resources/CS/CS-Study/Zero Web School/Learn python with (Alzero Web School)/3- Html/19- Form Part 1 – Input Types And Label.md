# <font color="#646a73">Form Part 1 – Input Types And Label</font>
---
### Input:
```html
<!doctype html>
<html>
  <head>
    <meta charset="UTF-8" />
    <title>Book Store</title>
    <meta name="description" content="This Is Our Book Store" />
  </head>
  <body>
    <div>
      <h1>Book Store</h1>
      <p>This Is My Book Store, Welcome</p>
    </div>
    <form>
      <div>
        <label>Username</label>
        <input type="text">
      </div>
      <br>
      <div>
        <label>Password</label>
        <input type="password">
      </div>
      <input type="submit">
    </form>
  </body>
</html>
```
